# Pèche ( heartopia.md )

## Notes

- on peut rester appuiyer sur le bouton pecher pour lancer plus précisement
- on ne peut pas pecher si ya pas d'ombre de poisson
- on peut utiliser des appat (3x tomates) pour attirer les poisson

- pecher dans l'evenement bateau permet d'xp plus rapidement
- peche lvl 3 => upgrade necessite 2 item upgrade

- le bateau 1 personne == 2min 45s
- bateau == 2000 pieces

- on vend les poisson à Albert Jr (vendeur de pièces d'or)
